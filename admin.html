⚠️ IMPORTANTE - Devi usare un server HTTP!

Non aprire i file direttamente (doppio click) perché Firebase non funziona con file://.

Come avviare il server:
  1. Apri il terminale nella directory del progetto
  2. Esegui: python3 -m http.server 8000
  3. Apri nel browser: http://localhost:8000/admin.html
Alternative:
  • Node.js: npx http-server -p 8000
  • PHP: php -S localhost:8000
  • VS Code: Estensione "Live Server"
